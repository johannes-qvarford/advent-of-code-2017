# Introduction

This is a solution to the [Advent of Code 2017 day 7 challenge](http://adventofcode.com/2017/day/7).

To try it out, execute the following from the root folder:

```shell
composer install
php main.php 7 res/d7_easy.txt
php main.php 7 res/d7_hard.txt
```

You can also use your own files as long as they follow the format presented in the challenge.
